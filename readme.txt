Keterangan Nomor 1

Input : 

Arah
Bergerak ke atas : Up Arrow
Bergerak ke kanan : Right Arrow
Bergerak ke kiri : Left Arrow
Bergerak ke bawah : Down Arrow

Color
Q : Black
W : Red
E : Green
R : Blue
T : Purple
Y : Yellow
U : Aqua

Speed
S : Stop
A : Faster
D : Slower

Mode
1 : Vertical Mirror
2 : Horizontal Mirror
3 : Vertical and Horizontal Mirror

Clear
c : clear
